Дискорд-Спаммер!
Небольшой дискорд спам бот, умеющий обходить медленный режим и предупреждения бота mee6!


Немного о функциях бота:

Функция "Количество сообщений" создает заданное количество сообщений.

Функция "Есть ли бот MEE6" создает сообщения со случайными цифрами. При использовании этой функции нельзя спамить выбранное сообщение!

Функция "Есть ли медленный режим" включает задержку 5 секунд на отправку сообщений!

Функция "Какой текст будем спамить" создает сообщения с введенным текстом!
